## MLflow Walkthrough:

MLflow is an open-source platform for managing the complete machine learning lifecycle — from experimentation to deployment.

It helps you:

- 📊 Track experiments – log parameters, metrics, models, and artifacts

- 📦 Package models – save models in a reproducible format

- 🗂 Manage versions – register and version models in a Model Registry

- 🚀 Deploy models – serve models via REST APIs or batch jobs

MLflow works seamlessly with libraries like Scikit-learn, TensorFlow, PyTorch, XGBoost, and more — making it framework-agnostic and production-friendly.

It has 4 main components:

- Tracking – Log runs, metrics, params

- Projects – Package ML code reproducibly

- Models – Standardized model format

- Model Registry – Versioning & lifecycle management
